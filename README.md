⚡ Android Performance Optimizer

> System performance optimization module for rooted devices (Magisk / KernelSU)

______________________________

 🚀 Key Features
- CPU scheduler optimization → improves responsiveness
- Reduced system latency (smoother UI)
- RAM optimization 
- I/O tuning
- Applies kernel and system tweaks


______________________________

 📦 Installation
1. Open **Magisk**
2. Go to **Modules**
3. Select **Install from storage**
4. Flash `sys_helper.zip`
5. Reboot the device


______________________________

 ⚠️ Warnings
- May cause bootloop if conflicting with kernel or other modules
- Not compatible with some custom ROMs
- Backup is recommended before use
